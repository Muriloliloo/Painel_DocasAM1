# Proxima etapa: Firebase em tempo real

Para transformar este painel em uma ferramenta ao vivo entre varios computadores, usar:

- Firebase Hosting ou GitHub Pages para hospedar o painel.
- Firebase Authentication para login.
- Firebase Firestore ou Realtime Database para sincronizar os dados.

## Perfis sugeridos

### Administrador

- Edita extracao.
- Edita base.
- Lanca rotas expedidas.
- Preenche justificativas.
- Limpa dados.
- Gera PDF.

### Leitor

- Visualiza painel de docas.
- Visualiza justificativas.
- Visualiza fechamento.
- Pode gerar PDF, se liberado.

## Dados que devem ir para o banco

- Data operacional.
- Total de rotas.
- Extracao.
- Base.
- Justificativas.
- Inicio e fim dos cronometros.
- Configuracoes do painel.
